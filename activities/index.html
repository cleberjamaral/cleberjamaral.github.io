Research

Publications:

  • Please check my recent publications in publications section.

Current projects:

  • 'RF-Abelhas' is a project in partnership with EPAGRI to remote monitoring of hives checking temperature, humidity, weight and other important parameters to improve honey productivity.
  • "Embedding Multi-Agent System Frameworks: A Benchmarking" is a research that aims to deploy different MAS frameworks in embedded platforms to check if it is feasible and check performance issues.
  • Revive Peleus project which is an implementation of an AgentSpeak agent interpreter based on the Jason implementation, with the addition of a STRIPS planning module. Currently there is a fork where the planning by jPlan (that uses graphplan) is working at least in simple tests.

Projects I want to develop/participate/supervise:

  • "Consulting Board": A MAS developing different agents to perform AI algorithms in specialized research fields. In an organizational dimension a coordination based in norms, trust and reputation may gather all opinions of these experts to suggest a decision.
  • Disaster-robotics-proalertas which aims to build an integrated system to control a team of heterogeneous robots and alert systems.
  • Ferramenta de testes de códigos C para uso didático is a tool for automatic test C codes to help students to get a prompt feedback and develop better codes. It uses gcc compiler and frama-c framework. Keywords: Programming, C language, Quality Assurance. Some insights in Formal verification A student code some related functions in TestadorC.

Extension

Finished projects:

  • Oficinas de Introdução à lógica de Programação (Parceria NAAHs) was a project of programming workshops for children (around 11 years old). It used Scratch platform as programming platform and Raspberry Pi as hardware platform. The project ocurred from 11/2016 to 11/2017 by IFSC's team formed by Cleber Jorge Amaral as Coordenator; Adriano Régis, Francisco Édson de Melo and Mauricio Edgar Stivanello as Supervisors; and Felipe D'Avila Mesquita as Student. From FCEE the team was formed by Sirlei Ignacio, Ana Paula and others. It also received help from the volunteer Isis and others.
  • Tutoria NAAH/S was a pilot project for tutoring of super gifted students. This time a child was supervised in project management and programming areas. A Project was choose and developed in 3 months (from 10/2016 to 12/2016). It was performed only by Cleber Jorge Amaral as Coordenador and Supervidor with help from FCEE people, specially, Sirlei Ignacio and Ana Paula.
